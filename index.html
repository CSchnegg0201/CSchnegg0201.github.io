What is Git?
Git is a system that keeps track of changes in different files and organizes work on files by multiple users.
Who started Git?
Git was started by the software engineer, Linus Torvalds
What is GitHub?
GitHub is an online hosting service that uses Git for version control.
What is a repository and its role?
A repository contains all of the files of a project. It stores a version history for each file. Other people can collaborate to the project online.
General thoughts on GitHub
GitHub is an excellent way for programmers to work together on a project over the internet.
